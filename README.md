# 🚀 Desafio Iniflex - Java

Projeto desenvolvido para o teste técnico, com foco em boas práticas, organização e testes automatizados.

---

## 📌 Tecnologias

- Java 21
- Spring Boot
- JUnit 5
- Mockito
- H2 Database
- Maven

---

## 📊 Funcionalidades

✔ Inserção de funcionários  
✔ Remoção do funcionário João  
✔ Formatação de datas e salários  
✔ Aumento de salário (10%)  
✔ Agrupamento por função  
✔ Listagem de aniversariantes (Outubro e Dezembro)  
✔ Identificação do funcionário mais velho  
✔ Ordenação alfabética  
✔ Soma total dos salários  
✔ Cálculo de salários mínimos  

---

## 🧪 Testes

- Testes unitários (Service)
- Testes de integração (Controller)
- Cobertura com JaCoCo (~77%)
